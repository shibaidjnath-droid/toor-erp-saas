# Instellingen Module – TOOR ERP

De instellingen bepalen dynamische configuraties zoals:
- Type Services
- Rollen
- Tags
- Email templates

## Type Services
Worden gebruikt bij contracten. Multi-select mogelijk.

## Rollen
- Schoonmaker → kan toegewezen worden aan planning
- Teamleider → krijgt toegang tot extra visualisaties (toekomst)
- Planner → mag planning wijzigen en genereren

## Tags
Vrij te definiëren klantlabels.

## Email Templates
Wordt gebruikt voor:
- Facturen
- Offertes (toekomst)
- Herinneringen (toekomst)

Placeholders:
- {{name}}
- {{customer}}
- {{invoiceNumber}}
- {{amount}}
- {{paymentUrl}}

## Veelgestelde vragen
### Waar wijzig ik rollen?
Instellingen → Rollen

### Hoe voeg ik een nieuwe service toe?
Instellingen → Service Types → + Nieuw

### Hoe wijzig ik email templates?
Instellingen → Email → Template Bewerken
